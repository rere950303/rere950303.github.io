## 절대경로
- html과 같은 웹 페이지나 파일이 가지고 있는 고유한 경로를 말한다.
- ex: /Users/hyungwook/Desktop/AbsoulutePath.txt

## 상대경로
- 현재 위치를 **기준**으로 정한 파일 등의 경로를 말한다.
	1. / : 루트 경로
	2. ./ : 현재 경로
	3. ../ :  상단 폴더의 경로
- ex: /Users/hyungwook/Desktop/ 위치에서
	- / : /Users/
	- ./ : /Users/hyungwook/Desktop/
	- ../ : /Users/hyungwook/
